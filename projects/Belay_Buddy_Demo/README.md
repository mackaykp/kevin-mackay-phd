# Belay Buddy Demo

This folder contains a browser-based demo of Belay Buddy that can run entirely as static files.

The demo shows the main user flow for organizing climbing rounds, including joining as host, managing participants, and stepping through upcoming rounds.

It is designed for presentation and portfolio use, not for production scheduling. Advanced backend pairing logic and real-time multi-user synchronization are intentionally not included.

To view it, open `index.html` in any modern web browser.
