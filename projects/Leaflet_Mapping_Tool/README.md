# Leaflet Mapping Tool

This project creates interactive maps that compare neighborhood income with access to green space in Canadian cities.

It combines Statistics Canada census data with OpenStreetMap green space data, then generates a bivariate map that helps users see where social and environmental advantages or gaps overlap.

The workflow includes data download, preparation, and map generation scripts, with outputs written as shareable HTML maps.

The main code is written in R, with a small Python script used for data retrieval.
