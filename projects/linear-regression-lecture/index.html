Guest lecture designed to make linear regression approachable for transportation geography students with widely varying statistics backgrounds.

Project Overview

Problem

The class included fourth-year geography students with very different quantitative backgrounds, from no statistics experience to statistics majors. Linear regression can feel overwhelming at first because students must learn new terms, notation, and assumptions quickly, especially without strong math confidence.

Context

While working at Statistics Canada, I was invited by a professor at McMaster University to deliver a linear regression lecture for a transportation geography course. The lecture needed to support mixed experience levels in one session while still preparing students to interpret and apply regression in course and real-world settings.

Goals

  • Demystify linear regression for beginners by making the model as visual and intuitive as possible.
  • Anchor the model in familiar high-school algebra (y = mx + b) to reduce cognitive load before introducing full statistical notation.
  • Build from core model meaning into prediction, residuals, model error, and R-squared.
  • Explain assumptions and exploratory checks with visual examples so students can assess whether a linear model is appropriate.

Stakeholders

  • Students: fourth-year transportation geography students with mixed quantitative preparation.
  • Professor: course alignment and instructional clarity within a single lecture slot.

Final deliverable

The final deliverable was a complete lecture slide deck (PDF) used in class, structured as a step-by-step progression from familiar algebra to formal linear regression, then into prediction, residuals, model fit, and assumptions.

Project Approach

Scope & Assumptions

  • Scope: One lecture focused on foundational linear regression concepts, model interpretation, and assumption-aware exploratory analysis.
  • Assumption: Teaching should prioritize conceptual clarity and visual explanation over heavy derivation so beginners can build confidence quickly.

Data Sources

  • Teaching materials: Slide visuals and worked examples designed to compare simple algebra and linear regression notation directly.
  • Analytical examples: Scatterplots and model outputs used to explain predictions, residuals, model error, and R-squared in plain language.

Analytical Design

  • Instructional sequence: start with y = mx + b -> map each component to the linear regression model -> interpret coefficients and predictions -> introduce residuals and model error -> explain R-squared -> review assumptions with visual exploratory checks.
  • Design choice: repeatedly switch between beginner-friendly algebra and full regression notation so students can see they represent the same underlying idea.

Workflow / Process

  1. Introduce why linear regression appears frequently in geography and transportation analysis.
  2. Build intuition through visual examples and algebra-to-regression translation.
  3. Walk through prediction, residuals, model error, and R-squared using incremental examples.
  4. Close with exploratory data analysis and model assumptions to support responsible model use.

Constraints & Considerations

  • Class time was fixed and student preparation varied substantially, so pacing needed to keep beginners engaged without losing advanced learners.
  • Statistical jargon and notation can create early barriers, so terminology had to be introduced gradually and paired with concrete visuals.
  • Assumptions and diagnostics can be abstract on first exposure, so examples needed to stay practical and easy to interpret.

Impact & Outcome

The lecture improved student confidence with linear regression by turning a potentially technical topic into a clear visual progression from familiar algebra to applied model interpretation. Student feedback shared with the course professor indicated that the session was easy to follow and increased confidence in using regression in future coursework and real-world contexts. Notably, students majoring in statistics also reported that the presentation was one of the clearest explanations of the material they had seen.
